## Installation

- Make sure you have NodeJS & NPM/Yarn installed on your computer, you can check using this command
  - `node -v` for NodeJS
  - `npm -v` for NPM
  - `yarn -v` for Yarn
- install all dependencies `npm install`
- copy .env.example file to .env
- fill the .env file using your api url and token credential
  - `VITE_API_KEY=APISTATICTOKEN`
  - `VITE_API_URL=APIURL`
- run project
  - `npm run dev`
